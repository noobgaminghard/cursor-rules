# Strict TypeScript

Zero-compromise TypeScript rules: no `any`, proper generics, exhaustive type safety.

## Covers
- Strict tsconfig settings (noUncheckedIndexedAccess, exactOptionalPropertyTypes)
- The "No Any" rule with alternatives (unknown, type guards, generics)
- Discriminated unions and type narrowing patterns
- Generic patterns (constraints, defaults, infer, mapped types)
- Utility type usage (Partial, Pick, Omit, Record, Extract, Exclude)
- Enum alternatives (as const objects, string literal unions)
- Exhaustive checks with assertNever
- Function overloads, parameter objects, readonly patterns

## Usage
```bash
cp .cursorrules /path/to/your/typescript-project/.cursorrules
```
