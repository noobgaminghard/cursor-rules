# Rust with Axum Web Framework

Cursor rules for building web services with Axum, Tokio, and the Rust async ecosystem.

## Covers
- Axum extractors (Json, Path, Query, State, Extension)
- Custom error types with IntoResponse
- Async patterns with Tokio (spawn, select!, join!)
- SQLx with compile-time checked queries
- Tower middleware layers (tracing, CORS, compression)
- Structured logging with tracing crate
- Testing with tokio::test and sqlx::test
- Memory safety patterns (Arc, borrowing, lifetimes)

## Usage
```bash
cp .cursorrules /path/to/your/rust-project/.cursorrules
```
