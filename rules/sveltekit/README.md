# SvelteKit with TypeScript

Cursor rules for building web applications with SvelteKit 2+ and TypeScript.

## Covers
- File-based routing (+page.svelte, +layout.svelte, +server.ts)
- Server-side load functions and form actions
- Svelte stores (writable, readable, derived)
- Component patterns (slots, events, actions, bindings)
- Hooks (handle, handleFetch, handleError)
- Scoped styling and Tailwind integration
- Testing with Vitest and Playwright
- Security (RLS, CSRF, environment variables)

## Usage
```bash
cp .cursorrules /path/to/your/sveltekit-project/.cursorrules
```
