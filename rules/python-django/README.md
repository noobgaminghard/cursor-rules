# Python Django 5+ with DRF

Cursor rules for building web applications with Django 5+ and Django REST Framework.

## Covers
- Model design (relationships, constraints, indexes, TimeStampedModel)
- Django REST Framework (ViewSets, Serializers, permissions)
- ORM best practices (select_related, prefetch_related, F/Q objects)
- URL routing with DefaultRouter and namespaces
- Authentication with SimpleJWT and custom permissions
- Testing with pytest-django and model-bakery
- Performance (N+1 prevention, caching, pagination)
- Security (CSRF, HSTS, ALLOWED_HOSTS)

## Usage
```bash
cp .cursorrules /path/to/your/django-project/.cursorrules
```
