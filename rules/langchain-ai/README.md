# LangChain / LLM Application Development

Cursor rules for building LLM-powered applications with LangChain, LangGraph, and related frameworks in Python.

## Covers
- LangChain Expression Language (LCEL) with pipe syntax
- Prompt engineering (ChatPromptTemplate, few-shot, system prompts)
- Structured output with Pydantic models
- RAG pipelines (chunking, embedding, retrieval, generation)
- Agents and tools (LangGraph, tool definitions, message passing)
- Memory and conversation management
- Error handling (rate limits, retries, fallbacks)
- Evaluation and testing with LangSmith

## Usage
```bash
cp .cursorrules /path/to/your/langchain-project/.cursorrules
```
