# Chrome Extension (Manifest V3)

Cursor rules for building Chrome extensions with Manifest V3, TypeScript, and modern web APIs.

## Covers
- Manifest V3 configuration (permissions, service workers, content scripts)
- Service worker patterns (event-driven, chrome.storage, chrome.alarms)
- Content scripts (DOM access, MutationObserver, Shadow DOM)
- Message passing (typed protocols, ports, cross-context communication)
- Storage (local, sync, session)
- Popup and options page development
- Security (CSP, minimum permissions, input validation)
- Testing with Jest/Vitest and Chrome DevTools

## Usage
```bash
cp .cursorrules /path/to/your/extension-project/.cursorrules
```
