# Node.js with Express and TypeScript

Cursor rules for building production-grade REST APIs with Express and TypeScript.

## Covers
- Layered architecture (Routes -> Controllers -> Services -> Repositories)
- Typed middleware and error handling
- Request validation with Zod
- Prisma ORM with repository pattern
- JWT authentication and RBAC
- Structured logging with Pino
- Testing with Vitest and Supertest
- Security (helmet, CORS, rate limiting, hpp)

## Usage
```bash
cp .cursorrules /path/to/your/express-project/.cursorrules
```
