# Docker and DevOps

Cursor rules for containerization, CI/CD pipelines, and infrastructure as code.

## Covers
- Dockerfile best practices (multi-stage builds, layer caching, non-root user)
- Docker Compose for development and production
- GitHub Actions CI/CD pipelines
- Infrastructure as Code (Terraform)
- Container security (scanning, minimal images, secrets management)
- Logging and monitoring (structured logging, metrics, alerts)
- Networking (custom bridges, reverse proxy, TLS)
- Environment management (12-factor app methodology)

## Usage
```bash
cp .cursorrules /path/to/your/project/.cursorrules
```
