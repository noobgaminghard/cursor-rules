# Go with Gin Framework

Cursor rules for building web applications and APIs with Go and the Gin framework.

## Covers
- Idiomatic Go patterns (error handling, naming, formatting)
- Gin handler/service/repository architecture
- Input validation with struct binding tags
- Database access (GORM, sqlx) with connection pooling
- Middleware (auth, logging, recovery, CORS, rate limiting)
- Structured logging with slog/zerolog
- Concurrency patterns (goroutines, channels, errgroup)
- Table-driven testing with testify

## Usage
```bash
cp .cursorrules /path/to/your/go-project/.cursorrules
```
