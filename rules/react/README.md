# React 18+ with TypeScript

Cursor rules for building React applications with TypeScript, functional components, and modern patterns.

## Covers
- TypeScript strict mode, component typing, event handler types
- Functional components with hooks (useState, useEffect, useMemo, useCallback, useReducer)
- State management (Context, Zustand, TanStack Query for server state)
- Component composition and custom hooks
- Error boundaries and async error handling
- Performance optimization (React.memo, lazy loading, virtualization)
- Accessibility (semantic HTML, ARIA, focus management)
- Testing with Vitest + React Testing Library

## Usage
```bash
cp .cursorrules /path/to/your/react-project/.cursorrules
```
